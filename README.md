# SQL-Project
SQL project to manage and analyze company data using MySQL.

## Corporate Data Management System (SQL Project)

## Overview
This project manages and analyzes company data such as employees, departments, salaries, attendance, and projects using SQL.It demonstrates skills in database creation, relational joins, subqueries, and analytical SQL functions.

## Tools Used
- MySQL  
- MySQL Workbench  

## Project Workflow
1. **Database Creation:** Created database `company_db` for managing company data.  
2. **Table Design:** Built tables - Department, Employee, Projects, Attendance, and Salary_History.  
3. **Data Insertion:** Inserted realistic company data for employees, departments, projects, attendance, and salary history.  
4. **Data Manipulation:** Used queries for `UPDATE`, `WHERE`, and `DELETE` operations.  
5. **Analysis Queries:** Applied aggregate, join, subquery, and window functions to extract insights.

## Key Insights
- **IT Department** has the highest average salary.  
- Reena is the top-paid employee with ₹80,000.  
- **Operations projects** have the biggest budgets.  
- Employee attendance shows most are present on weekdays.  
- Average project budget across departments is **₹1.2L.**
